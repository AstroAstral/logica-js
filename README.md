# Lógica de Programação com JavaScript 

Este repositório contém meus estudos iniciais sobre fundamentos de JavaScript, focando em boas práticas, tipos de dados e controle de fluxo. 

## O que eu pratiquei:
- **Variáveis e Escopo:** Diferença prática entre `var`, `let` e `const`.
- **Tipagem:** Manipulação de strings, números e booleanos.
- **Template Strings:** Interpolação de texto de forma moderna.
- **Estruturas Condicionais:** Lógica de tomada de decisão com `if/else`.

## Tecnologias Utilizadas:
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)
- [Node.js](https://nodejs.org/) (Ambiente de execução)
- [Git](https://git-scm.com/) & [GitHub](https://github.com/) (Versionamento)

## Como rodar o projeto:
1. Clone este repositório.
2. Certifique-se de ter o Node.js instalado.
3. No terminal, execute o comando:
   ```bash
   node exercicios.js
